Authentication Mode

Choose how users will access Huntarr.

Login Mode
Standard login required for all connections. Recommended for most users.
Local Bypass Mode
Local network connections (192.168.x.x, 10.x.x.x) bypass login. External access still requires authentication.
No Login Mode
Completely disable authentication. Only recommended when behind a secured reverse proxy.
Warning: Only use No Login Mode if your reverse proxy (e.g., Cloudflare, Nginx) is properly securing access!
