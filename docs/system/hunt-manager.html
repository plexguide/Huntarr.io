Hunt Manager

Monitor and manage all active hunting operations across your apps.

Overview

  • The Hunt Manager provides a centralized view of all hunting activity
  • See what's currently being searched, queued, or completed
  • Monitor schedule execution and timing
  • Track processed media across all apps

Dashboard View

The Hunt Manager shows:

  • Active Hunts: Currently running search operations
  • Queue: Items waiting to be processed
  • Recent Activity: Last completed hunt cycles with results
  • Statistics: Total items processed, success rates, timing data

Per-App Status

Each connected app shows:

  • Hunt cycle status (running, idle, waiting)
  • Last cycle time and next scheduled cycle
  • Items processed in the current/last cycle
  • Missing/upgrade counts

Processed Media Tracking

  • Huntarr tracks all media it has processed to avoid duplicate searches
  • State management modes: Database (default) or API-based
  • Reset processed state if you want Huntarr to re-search items

Tips

  • Check the Hunt Manager if hunts seem stuck or slow
  • The statistics help identify which apps need schedule tuning
  • Use this page to verify schedules are running at the expected times
  • If an app shows no activity, verify the instance connection and schedule
