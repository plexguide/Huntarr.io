NZB Hunt

Search Usenet indexers directly from Huntarr.

What is NZB Hunt?

  • NZB Hunt is Huntarr's built-in Usenet search tool
  • Search across multiple Newznab-compatible indexers from one interface
  • Browse results, view details, and send NZBs to your download client
  • Currently in Beta

Features

  • Multi-indexer search: query multiple indexers simultaneously
  • Category filtering: Movies, TV, Music, Books, etc.
  • Results sorting: by age, size, grabs, or indexer
  • Direct download: send NZBs to SABnzbd or NZBGet with one click
  • Search history: view past searches

Getting Started

  1. Navigate to NZB Hunt in the sidebar
  2. Add Usenet indexers (must be Newznab-compatible)
  3. Configure your download client
  4. Start searching!

Adding Indexers

  • Go to NZB Hunt > Settings > Indexers
  • Click "Add Indexer"
  • Enter the Newznab URL and API key
  • Common indexers: NZBGeek, DrunkenSlug, NZBFinder, etc.
NZB Hunt only supports Newznab-compatible indexers (Usenet). For torrent searching, use Movie Hunt or your *arr apps.

Search Tips

  • Use specific search terms for best results
  • Category filtering helps narrow results
  • Check multiple indexers if one doesn't return results
  • Older content may not be available on all indexers due to retention limits
