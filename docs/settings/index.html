Huntarr Settings

In Huntarr, open Settings from the main sidebar. The settings sidebar then shows these sections:

Settings sections

Main

General options: timezone, check for updates, debug mode, display resources, low usage mode, authentication, SSL verify, TMDB cache, base URL, and per-instance options when you edit an app.

View Main Settings →

Scheduling

Configure when Huntarr runs its cycles and how often each app is processed.

View Scheduling →

Notifications

Apprise-based notifications for missing and upgrade events. Configure URLs, levels, and per-app options.

View Notifications →

Backup / Restore

Automatic and manual database backups, restore from backup, and download/upload backup files.

View Backup & Restore →

Logs

Log viewer and log-related settings (e.g. refresh interval) from within Settings.

Settings doc (Logs) →

User

User profile, password change, and two-factor authentication (2FA).

View Security / User →
