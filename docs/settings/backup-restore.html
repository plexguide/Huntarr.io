Backup & Restore

Huntarr can back up and restore its databases (main, logs, manager) from Settings.

Backup Frequency

How often automatic backups are created (in days). Default: 3. Backups are stored in the same directory as the database.

Backup Retention

Number of backup copies to keep. Older backups are deleted automatically. Default: 3.

Restore Database

Select a backup from the list and confirm with "RESTORE" to overwrite the current database with the selected backup. This cannot be undone.

Download Backup

Select a backup and download it as a ZIP file for safekeeping or moving to another instance.

Upload Backup

Upload a ZIP file containing a valid Huntarr backup to restore from (e.g. after downloading from another instance).
