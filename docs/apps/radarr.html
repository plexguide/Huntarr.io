Radarr

Hunt for missing movies and automatically upgrade quality.

Overview

  • Radarr manages movies — Huntarr connects to find missing films and upgrade quality
  • Works with Radarr v3+
  • Multiple Radarr instances supported

Instance Settings

URL, API key, name (same pattern)

Hunt Settings

SettingDefaultDescription

Hunt UpgradesEnabledUpgrade movies below cutoff quality

Minimum AvailabilityAnnouncedWhen a movie becomes searchable (Announced, In Cinemas, Released)

How Hunting Works

Similar to Sonarr but for movies. Steps:

  1. Huntarr queries Radarr's API for missing or upgradeable movies
  2. Filters by monitored status, minimum availability, and quality cutoff
  3. Triggers a search command in Radarr
  4. Radarr handles downloading via its configured indexers
  5. Repeats on schedule

Tips

  • Minimum Availability is important — searching for "Announced" movies rarely finds results
  • Use "Released" for the best hit rate
  • Radarr's custom formats can help Huntarr find the best quality
  • Multiple instances let you maintain separate quality libraries (e.g., 1080p + 4K)
If movies aren't being found, check that Radarr's Minimum Availability matches your expectations. A movie marked "Announced" may not have any releases yet.
