App Connections

Connect and manage your *arr instances with Huntarr.

How App Connections Work

  • Huntarr communicates with each *arr app via its API
  • You provide the app URL and API key
  • Huntarr monitors missing/upgradeable media and triggers searches
  • Each instance is managed independently with its own settings

Adding an Instance

  1. Navigate to the Apps page from the sidebar
  2. Select the app type (Sonarr, Radarr, etc.)
  3. Click "Add Instance"
  4. Enter a friendly name, the app URL, and API key
  5. Click "Test Connection" to verify
  6. Save the instance
Your API key is found in each *arr app under Settings > General > Security.

Managing Instances

  • Edit instance settings (URL, API key, name)
  • Enable/disable specific instances without deleting
  • Configure per-instance hunt settings
  • Delete instances you no longer need

Supported App Grid

Sonarr

TV series — hunt missing episodes and upgrade quality.

Radarr

Movies — hunt missing films and upgrade quality.

Lidarr

Music — hunt missing albums and tracks.

Readarr

Books and audiobooks — hunt missing content.

Whisparr

Adult content — hunt missing and upgrade.

Prowlarr

Indexer management — sync with Huntarr.

Swaparr

Stalled download management and swap-outs.

SettingDescription
NameA friendly label for this instance
URLFull URL including port (e.g., http://localhost:8989)
API KeyFound in Settings > General of the app
EnabledToggle hunting for this instance on/off
Hunt MissingSearch for missing media
Hunt UpgradesSearch for quality upgrades
