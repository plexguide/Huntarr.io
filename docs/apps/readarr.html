Readarr

Hunt for missing books and audiobooks automatically.

Overview

  • Readarr manages books and audiobooks
  • Huntarr connects to find missing titles and upgrade quality
  • Works with Readarr v0.1+
  • Multiple instances supported (e.g., ebooks vs audiobooks)

Configuration

Instance Settings

Same as general app settings (URL, API key, name)

Hunt Settings

SettingDefaultDescription
Hunt MissingEnabledSearch for books without files
Hunt UpgradesEnabledUpgrade books below cutoff quality
Missing Items per Cycle1Books to search per cycle
Upgrade Items per Cycle1Books to try upgrading per cycle
Monitored OnlyEnabledOnly hunt monitored authors/books

How Hunting Works

  1. Huntarr queries Readarr's API for missing or upgradeable books
  2. Filters by monitored status
  3. Triggers a book search in Readarr
  4. Readarr handles download
  5. Repeats on schedule

Tips

  • Book indexers can be limited — patience and multiple indexers help
  • Audiobook searches may require specialized indexers
  • Separate instances for ebooks and audiobooks is recommended
  • Check Readarr's Activity queue for search progress
