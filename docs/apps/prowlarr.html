Prowlarr in Huntarr

Prowlarr is an indexer manager for the *arr stack. In Huntarr you add a single Prowlarr instance; your Sonarr, Radarr, Lidarr, Readarr, Whisparr, and Eros instances use Prowlarr’s indexers for searches, so you manage indexers in one place.

Adding Prowlarr

Go to Apps in the sidebar and open Prowlarr. Add your Prowlarr URL and API key (from Prowlarr’s Settings → General). Huntarr does not run “missing” or “upgrade” cycles for Prowlarr; it only connects so that other apps can use your Prowlarr-configured indexers when Huntarr triggers searches.

Single instance

Huntarr supports one Prowlarr instance. Configure all your indexers in Prowlarr and sync them to your *arr apps; Huntarr will use those indexers when it triggers searches in Sonarr, Radarr, and the others.
