Profiles & Custom Formats

Control quality preferences and release scoring for Movie Hunt.

Quality Profiles

Quality profiles define what Movie Hunt considers acceptable quality.

Default Profile

Movie Hunt ships with a default profile. You can customize:

  • Minimum quality (e.g., 720p, 1080p, 2160p)
  • Preferred quality (what to aim for)
  • Cutoff quality (stop upgrading beyond this)
  • Allowed formats (H.264, H.265, AV1, etc.)

Custom Formats

Custom formats let you score releases based on specific attributes.

How Scoring Works

  • Each release is scored against your custom formats
  • Higher scores are preferred
  • A minimum score can be set — releases below it are rejected
  • Movie Hunt shows the score and breakdown on the movie detail page

Creating a Custom Format

  1. Go to Movie Hunt > Settings > Custom Formats
  2. Click "Add Custom Format"
  3. Define the name and conditions (regex match on release name, etc.)
  4. Assign a score
  5. Save

Minimum Availability

Controls when Movie Hunt considers a movie available for searching:

  • Announced: Search as soon as announced (may yield no results)
  • In Cinemas: Search once the movie is in theatres
  • Released: Search only after home release (recommended for best results)
Setting minimum availability to 'Announced' will cause many failed searches. For the best experience, use 'Released' or 'In Cinemas'.
