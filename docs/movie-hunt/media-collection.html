Media Collection

Browse, search, and manage your Movie Hunt library.

Browsing Movies

  • Movie Hunt displays your collection in a grid or list view
  • Each movie shows: poster, title, year, quality status, custom format score
  • Filter by status: All, Missing, Downloaded, Requested

Movie Detail View

Clicking a movie opens its detail page with:

  • Full metadata (title, year, overview, runtime, genres)
  • Quality and format information (resolution, codec)
  • Custom format score with breakdown tooltip
  • Download history (all past grab and import attempts)
  • Action buttons: Force Search, Force Upgrade, Edit, Delete

Adding Movies

  • Use the search/discover feature to find new movies
  • Search by title, TMDb ID, or browse trending/popular
  • Click "Add to Library" to monitor a movie
  • Once added, Movie Hunt will search for it based on your profile settings

Requesting Movies

  • Requests integration lets you request movies for your library
  • Requested movies appear in Movie Hunt with "Requested" status
  • Once found and downloaded, the status changes to "Downloaded"

File Management

  • View the full file path of downloaded movies
  • Delete movies and their files from the detail view
  • Edit movie settings (monitored status, quality profile, minimum availability)
